Whot2Watch Branding

Core identity
- Name: Whot2Watch
- Voice: concise, confident, helpful
- Vibe: premium, modern, cinematic

Colors
- Primary gradient: #6AE3FF → #A78BFA (energy + premium)
- Background (dark): #0B0F14; (light): #F7F8FA
- Text (dark): #E6EDF3; (light): #0B1220
- Border: #1D2330 (dark), #E5E7EB (light)

Typography
- Sans: Geist Sans (system fallback: Inter/system-ui)
- Mono (tech accent only): Geist Mono

Logo
- Mark: rounded square with gradient; monogram W chevrons + play triangle + spark
- Usage: 24–128px sizes, gradient background preferred, keep clear space = 1× logo corner radius

UI tokens (web)
- --radius: 12–16px
- Gradient text for titles and brand-name
- Elevated surfaces use subtle border and shadow on hover

Do’s
- Use gradient sparingly for emphasis (brand-name, section titles, CTAs)
- Maintain strong contrast for accessibility

Don’ts
- Don’t overcrowd with multiple gradients on the same block
- Don’t reduce logo below 16px effective size
