# Architecture Decision Records (ADRs)

Use ADRs to document significant architectural or process decisions.

## How to add a new ADR

1. Copy `0000-template.md` to `NNNN-descriptive-title.md` with the next sequential number.
2. Fill in Context, Decision, Consequences, and Alternatives.
3. Submit in the same PR as the implementation whenever feasible.

References: https://adr.github.io/
